# Sym
Symbolic computation, symbolic regression, AI. By SymbolicComputation.com.

[Sym help](sym.md)

[SymAI help](symai.md)

Sym is under construction. This project is at the early alpha stage, there will be bugs and breaking changes at this point.

I built a crude browser based page, it is [here](https://symonlineserver20200716184959.azurewebsites.net/). The SymApp is multithreaded and faster. There are also far more features. The browser based calculations use Blazor, which is interesting as you can effectively run C# on the client browser.

Keywords: Computer Algebra System, C#, .NET Core
